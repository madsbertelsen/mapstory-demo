How to embed

In order to imbed this route you should add 2 tags to you html like this: 
<!DOCTYPE html>
    <html>
      <body>
        ...
        <div data-mapstory-route-id="237"></div>
        ...
        <script src="https://mapstory-deployments.s3.eu-central-1.amazonaws.com/hoejskoleruten/v2/bundle.js"></script>
      </body>
    </html>>
